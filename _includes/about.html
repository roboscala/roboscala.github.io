About

sbt-robovm is a plugin for the Scala build tool that aims to make it as simple as possible to build Scala applications for iOS and OSX using RoboVM, which provides a compiler and interfaces to Objective-C code and APIs.

RoboVM includes many samples to help you get up and running, and we provide some of those examples ported to Scala. On top of that, you can find tutorials below on various subjects specifc to working with our tools.
